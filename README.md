## 🚀 Installation et Déploiement

### Prérequis
- Un navigateur web moderne
- Un compte GitHub (pour héberger le code)
- Un compte Vercel (pour le déploiement)

### Option 1 : Déploiement Simple sur Vercel

1. **Cloner le repository**
```bash
git clone https://github.com/VOTRE_USERNAME/portfolio.git
cd portfolio
```

2. **Déployer sur Vercel**
```bash
# Installer Vercel CLI (si pas déjà fait)
npm i -g vercel

# Déployer
vercel
```

3. **Suivre les instructions**
- Sélectionner le projet
- Configurer les paramètres
- Déployer !

### Option 2 : GitHub Pages

1. **Activer GitHub Pages**
   - Aller dans Settings > Pages
   - Source : Deploy from branch
   - Branch : main / root
   - Save

2. **Accéder au site**
   - URL : `https://VOTRE_USERNAME.github.io/portfolio/`

### Option 3 : Développement Local

Ouvrez simplement `index.html` dans votre navigateur :
```bash
# Méthode 1 : Double-clic sur index.html

# Méthode 2 : Serveur local Python
python -m http.server 8000

# Méthode 3 : Serveur local Node.js
npx serve

# Puis visiter : http://localhost:8000
```

## ⚙️ Personnalisation

### Modifier les Couleurs

Changez les variables CSS dans `index.html` :
```css
:root {
  --bleu: #1e7fff;      /* Couleur principale */
  --cyan: #00e5c3;      /* Couleur secondaire */
}
```

### Ajouter des Projets

Dupliquez une carte dans la section `#portfolio` de `index.html`
